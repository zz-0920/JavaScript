# 异步
- 进程
- 线程

1. 在不同的场景中进程都可以用来描述该场景中的一个效果，而线程则是用来描述进程中的一个效果。通常多个线程配合工作构成一个进程
2. v8 运行一份代码，会创建一个进程，从上往下执行代码，遇到同步代码就直接执行，遇到异步代码就先跳过，先去执行后面的同步代码，等到后面的同步代码全部执行完毕后再执行异步代码
3. JS默认是单线程的语言：因为 js 设定是为了做浏览器的脚本语言，尽量少的开销用户设备的性能
