# 服务端逻辑分层
1. 路由层: 处理前端的请求，根据请求的路径和方法，调用对应的业务逻辑
2. 控制层: 执行响应逻辑时，调用服务层的方法，返回响应体
3. 服务层: 处理业务逻辑，调用数据层的方法，返回数据
4. 数据层: 处理数据的读写操作，返回数据
