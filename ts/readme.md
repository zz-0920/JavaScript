# TypeScript
1. 静态类型：声明一个变量时要指定类型
2. 类型推断：当变量被赋值时，TS 会根据赋值自动推断变量的类型

## 类型
- any - 任意类型
- never - 永远不会有值
- unknown - 未知类型

- string - 字符串 String
- number - 数字 Number
- boolean - 布尔 Boolean
- null - 空 Null
- undefined - 未定义 Undefined
- bigint - 大整数 BigInt
- symbol - 符号 Symbol

- Object - 广义对象类型，约等于 any
- object - 狭义对象类型，只包含 对象，数组，函数

- type - 类型别名
- interface - 接口
- enum - 枚举
